virtual-host-script_v1 - Script to create local host for Apache version < 2.4

This script creates virtual hosts for apache version < 2.4 (tested in 2.2.22)

Usage

Give 'execute' permission to the script file : sudo a+x virtual-host-script_v1
sudo ./virtual-host-script_v1 domain-name webroot

Eg: sudo ./virtual-host-script_v1 test /home/Project/test

This will create a virtual host test.local
